# Pokédex (Kanto)

Projeto de Pokédex em desenvolvimento que recebe o número de um Pokémon digitado pelo usuário e devolve o nome correspondente.

> 🚧 Em desenvolvimento — feito como parte do meu aprendizado de JavaScript.

## Como rodar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Digite o número do Pokémon no campo.
4. Clique em "procurar".
5. O nome do Pokémon aparece na tela.

## O que já funciona

- Busca o nome do Pokémon a partir do número digitado (Pokémons 1 a 4, por enquanto).
- Resultado exibido diretamente na página (sem precisar abrir o console).
